BotUp Chatbot Test Page

This page demonstrates how to integrate the BotUp chatbot widget into your website.

Embed Code

Add the following code to your website to embed the BotUp chatbot widget:

<script src="http://localhost:5000/widget.js" data-botid="YOUR_BOT_ID"></script>

Replace YOUR_BOT_ID with your actual bot ID.
